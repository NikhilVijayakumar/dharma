# Product Track Separation Protocol

- Never mix children's educational content with mature fiction outputs.
- Every workflow must classify track context before execution.
- Cross-track reuse requires explicit policy and reviewer approval.
- Automated trigger: any missing or conflicting track classification blocks execution.
- Escalation threshold: first detected track-mixing event opens compliance incident and freezes release path.
- Required evidence: reviewer id, approval timestamp, and exception rationale for all cross-track approvals.
