# Mandatory Persona Documentation Protocol

- No persona activation without approved persona document and domain boundaries.
- Persona document must include scope, excluded domains, tone constraints, and escalation routes.
- Any persona update requires reviewer identity, rationale, and effective timestamp.
- Activation gate fails closed when documentation is stale, incomplete, or unapproved.
- Store persona document checksum and approval trace in compliance audit trail.
