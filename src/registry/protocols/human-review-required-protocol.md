# Human Review Required Protocol

- Human review is mandatory before publication or irreversible rollout.
- Any low-confidence or high-uncertainty action must escalate.
- Approval identity and rationale must be logged for traceability.
- Publication is blocked when reviewer identity, timestamp, or rationale is missing.
- Any critical-output bypass attempt triggers compliance incident escalation.
- Critical threshold: 0 unauthorized publication events; warning threshold: any incomplete sign-off metadata.
